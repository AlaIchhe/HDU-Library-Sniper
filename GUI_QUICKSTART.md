## ✨ 新增功能

### 1. 方案创建 ✓
- 点击"创建方案"按钮
- 选择房间类型 → 自动加载楼层
- 选择楼层 → 显示可用座位号
- 输入座位号和时间参数
- 实时验证，一键创建

### 2. 方案删除 ✓
- 点击"删除方案"按钮
- 多选要删除的方案（Ctrl/Shift）
- 二次确认，安全删除

### 3. 批量修改时间 ✓
- 点击"批量修改时间"按钮
- 多选要修改的方案
- 输入新值（留空保持原值）
- 一次修改多个方案

### 4. 房间浏览 ✓
- 点击"浏览房间"按钮
- 选择房间类型
- 查看楼层、座位数、可用座位号
- 只读浏览，方便选座

## 📋 操作流程示例

### 首次使用
1. 启动 GUI：`python main.py --gui`
2. 认证标签 → 输入学号密码 → 登录
3. 方案管理 → 创建方案
   - 选择"自习室"
   - 选择楼层（例如"A区1楼"）
   - 输入座位号（例如"101"）
   - 设置时间：13:00 起 9 小时，提前 1 天
   - 点击确定
4. 方案列表自动刷新，显示新方案
5. 抢座标签 → 输入时间（例如"23:59:59"）→ 开始抢座

### 日常使用
1. 启动 GUI（自动认证）
2. 方案管理 → 刷新查看方案
3. 抢座 → 设置时间 → 开始

## 🔍 对话框功能

### 创建方案对话框
- 房间类型下拉选择
- 楼层级联加载（异步）

- SpinBox 范围约束
- 实时验证，动态禁用 OK 按钮

- 多选列表（ExtendedSelection）
- 显示状态（✓/✗）、房间、座位、时间

### 批量修改时间对话框

- 三个输入框（开始小时、时长、天数）
- 留空保持原值

- 确认对话框显示修改详情

### 房间浏览对话框
- 房间类型选择
- 表格显示（楼层 ID、名称、座位数、座位号）
- 座位号格式化（前 10 个 + 总数）
- 只读模式

## 🛠️ 技术特性

- **异步加载**: 所有网络请求在后台线程执行
- **实时验证**: 输入变化即时反馈
- **级联更新**: 房间类型 → 楼层 → 座位提示
- **错误处理**: 友好的错误提示，不崩溃
- **用户反馈**: 状态栏、消息框、进度对话框

## 🐛 故障排除

### GUI 无法启动
```bash
# 确认 PySide6 已安装
pip install PySide6

# 测试模块导入
python test_gui.py
```

### 对话框报错
- 检查网络连接
- 确认已登录认证
- 查看错误对话框的详细信息

### 方案创建失败
- 确认座位号存在（参考座位号提示）
- 检查参数范围（0-23, 1-15, 0-7）
- 查看验证错误提示
