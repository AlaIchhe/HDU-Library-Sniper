## 📝 技术说明

### 后台守护进程模式

**触发命令**:
```bash
python main.py --daemon    # 或
python main.py --run-now   # 兼容旧脚本
```

**执行流程**:
1. 读取配置文件（plans.yaml, credentials.yaml）
2. 初始化服务层
3. 尝试缓存登录，失败则用凭据续登
4. 读取启用的方案
5. 执行抢座（重试、窗口轮询）
6. 推送通知
7. 记录日志
8. 退出（返回退出码）

**特点**:
- ✅ 完全非交互
- ✅ 不加载 GUI 依赖（轻量级）
- ✅ 适合系统定时任务
- ✅ 明确的退出码（0=成功, 1=失败, 2=认证失败, 3=无方案）

### 系统定时任务配置

**Windows (任务计划程序)**:
```bash
schtasks /Create /TN "HDU-Library-Sniper-Daily" \
  /TR "python.exe C:\path\to\main.py --daemon" \
  /SC DAILY /ST 23:59:55 /F
```

**Linux (crontab)**:
```bash
59 23 * * * /usr/bin/python3 /path/to/main.py --daemon
```
