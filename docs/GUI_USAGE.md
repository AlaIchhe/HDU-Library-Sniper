# GUI 使用说明

## 启动 GUI

### 方法 1: 使用 Python 命令
```bash
python main.py --gui
```

### 方法 2: 使用快捷脚本
- **Windows**: 双击 `start_gui.bat`
- **Linux/macOS**: 运行 `bash start_gui.sh`

## 界面介绍

GUI 主窗口包含三个标签页：

### 1. 认证标签页

**功能**：
- 输入学号和密码
- 点击"登录"按钮进行认证
- 认证状态实时显示
- 凭据自动保存到 `data/credentials.yaml`

**使用流程**：
1. 首次使用时，输入学号和密码
2. 点击"登录"按钮
3. 等待后台 Playwright 浏览器完成 SSO 登录
4. 登录成功后，凭据会自动保存，下次启动自动认证

### 2. 方案管理标签页

**功能**：
- 查看所有预约方案
- 显示方案的启用状态
- 刷新方案列表

**当前状态**：
- 方案创建功能正在开发中
- 目前请通过以下方式管理方案：
  - 使用终端交互模式：`python main.py`
  - 直接编辑 `data/plans.yaml` 文件

**方案显示格式**：
```
1. [✓] 自习室 - 座位 A001
   时间: 13:00 起，9h，提前 1 天

2. [✗] 阅览室 - 座位 B202
   时间: 9:00 起，6h，提前 1 天
```
- `✓` 表示已启用
- `✗` 表示未启用

### 3. 抢座标签页

**功能**：
- 立即抢座或定时抢座
- 实时倒计时显示
- 详细执行日志
- 任务取消功能

**使用流程**：

#### 立即抢座
1. 执行时间输入框留空
2. 点击"开始抢座"按钮
3. 查看执行日志，等待结果

#### 定时抢座
1. 在执行时间输入框输入时间，格式：
   - `HH:MM` (例如: `23:59`)
   - `HH:MM:SS` (例如: `23:59:59`)
2. 点击"开始抢座"按钮
3. 观察倒计时显示
4. 到达指定时间后自动开始抢座

#### 取消任务
- 在任务执行过程中，点击"取消"按钮可以中止任务

**日志说明**：
- `✓` 表示该方案成功
- `✗` 表示该方案失败
- 日志会实时显示每次尝试的结果

## 重试参数

GUI 使用 `config/config.yaml` 中的默认重试参数：
- `max_trials`: 最大重试次数
- `retry_delay`: 重试基础延迟（秒）
- `window_wait_seconds`: 预约窗口等待超时（秒）
- `window_poll_interval`: 预约窗口轮询间隔（秒）

如需调整，请编辑配置文件或使用终端交互模式的定时预约功能（可临时覆盖）。

## 常见问题

### Q: GUI 启动后没有窗口显示？
A: 检查是否有错误输出，确保 PySide6 已正确安装：
```bash
pip install PySide6
```

### Q: 登录一直失败？
A: 确保：
1. 学号和密码正确
2. Playwright 浏览器已安装：`playwright install chromium`
3. 网络连接正常
4. 查看认证状态区域的详细错误信息

### Q: 没有方案可以选择？
A: 需要先创建预约方案：
- 使用终端模式：`python main.py`
- 选择"创建预约方案"
- 或直接编辑 `data/plans.yaml`

### Q: 如何关闭 GUI？
A: 点击窗口右上角的关闭按钮，如果有任务正在执行，会弹出确认对话框。

## 技术细节

### 异步处理
- 所有阻塞操作（登录、抢座）都在独立的 QThread 中执行
- 主界面保持响应，不会卡顿
- 通过信号/槽机制实时更新界面

### 数据文件
- `data/credentials.yaml`: 保存的学号和密码（已 gitignore）
- `data/session.cache`: 登录态缓存
- `data/plans.yaml`: 预约方案配置

### 日志
- GUI 日志显示在界面上
- 详细日志仍然会写入 `logs/` 目录
- 通知推送（如配置）也会正常工作
