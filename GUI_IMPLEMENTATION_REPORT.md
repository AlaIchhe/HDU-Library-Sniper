## 🎨 技术亮点

### 1. 异步加载
- 所有网络请求在 `LoadFloorsWorker` 中执行
- 显示 `QProgressDialog` 加载提示
- 支持用户取消操作
- 界面永不卡顿

### 2. 级联加载
- 房间类型改变 → 自动加载对应楼层
- 楼层改变 → 自动更新座位号提示
- 数据按需加载，避免一次性加载全部

### 3. 实时验证
- 输入变化时触发 `_validate_and_update_ui()`
- 动态控制 OK 按钮启用/禁用
- 即时显示错误提示

### 4. 部分更新
- 批量修改时间支持留空保持原值
- 使用 `**kwargs` 只传递非空字段
- 复用 `PlanRepository.batch_set_time()` 的 `None` 检测逻辑

### 5. 错误处理
- 网络错误：显示详细错误，不关闭对话框
- 验证错误：高亮显示，阻止提交
- 回退警告：fell_back=True 时显示黄色提示
- 异常捕获：所有异常都有友好的错误对话框

### 6. 用户体验
- 二次确认删除操作
- 显示操作结果（"已删除 N 个方案"）
- 状态栏实时反馈
- 多选提示（Ctrl/Shift 键）
- 座位号提示（前 10 个 + 总数）
